# 🧪 Flask: CRUD & API (Estudos do Framework)

Este repositório foi desenvolvido como um  de estudos para dominar o framework **Flask**. O objetivo principal foi construir uma aplicação que lida com persistência de dados, organização de rotas e consumo de serviços externos.


## 🚀 O que este projeto faz?
A aplicação consiste em um sistema de gerenciamento (CRUD) para livros que permite criar, ler, atualizar e excluir registros de forma dinâmica, utilizando uma interface web conectada a um banco de dados local.
Nele também estamos conseguindo a API da themoviedb para fazer a listagem de filmes e criando uma rota dinamica para listar eles.

## 🛠️ Tecnologias e Ferramentas
* **Python 3.x**
* **Flask** (Micro-framework web)
* **Flask-SQLAlchemy** (ORM para manipulação do SQLite)
* **SQLite** (Banco de dados relacional leve)
* **Jinja2** (Engine de templates para o Front-end)

## 🧠 Aprendizados Consolidados
Durante o desenvolvimento, foquei em resolver problemas reais de aplicações web:
* **Operações CRUD:** Implementação da lógica de persistência de dados no SQLite.
* **Paginação de Dados:** Organização de grandes volumes de registros em múltiplas páginas para melhorar a experiência do usuário (UX).
* **Gerenciamento de Rotas:** Criação de caminhos dinâmicos e manipulação de métodos HTTP (GET, POST, etc.).
* **Integração com APIs:** Consumo de APIs externas para enriquecer as funcionalidades da aplicação.
